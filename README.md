# k8-vmware-visualisation

# Getting Started

### Requirements
- Python >= 3
- Pip
- Install Pyvmomi

```sh
$ pip install pyvmomi
```

### Setup and Usage

```sh
$ git clone https://github.com/k8-proxy/k8-vmware-visualisation

$ cd k8-vmware-visualisation

$ ./upwork-devs/Lwasampijja-Baker/getVMs.py > file.json
```
